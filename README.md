# Heart Disease Risk Calculator

This project builds a machine learning model to predict heart disease risk using clinical indicators.

## Models Used
- Logistic Regression
- Random Forest

## Performance
Logistic Regression:
- Accuracy: 90.7%
- AUC: 0.947

## Features Used
- Age
- Blood Pressure
- Cholesterol
- Chest Pain Type
- ST Depression
- Number of Major Vessels
- Thallium Test

## App
The project includes a Streamlit web application that allows users to input clinical features and estimate heart disease risk.
